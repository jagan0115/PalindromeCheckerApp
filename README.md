# PalindromeCheckerApp

## Objective
The objective of this project is to check whether a given string or number is a palindrome or not using Java.

## Project Flow
1. User gives input (string or number).
2. The input is reversed using logic.
3. Original input is compared with reversed input.
4. Result is displayed as Palindrome or Not Palindrome.

## Use Case 1
- Input is taken as a string.
- The program checks if the string is same when reversed.
- If both are equal, it prints Palindrome.
